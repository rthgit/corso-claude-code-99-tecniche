# Claude Code — 99 Techniques
### Open source educational repository, operational collection, and continuous lab on workflow, automation, and agentic coding

This repository collects the material for the **Claude Code — 99 Techniques** course and is designed as a **base for study, consultation, and practical experimentation**.

It is not just a linear educational support but an **evolving technical collection**: an organized archive of techniques, patterns, workflows, configurations, operational practices, and approaches emerged from the real-world use of Claude Code.

The goal is three-fold:

- To offer a **structured educational path**;
- To maintain an **updated collection of techniques and news**;
- To build a repository useful over time as a **practical knowledge base**.

This repo can therefore be read in three distinct and complementary ways:

- As a **proper course**;
- As a **consultable operational manual**;
- As an **open and updatable collection** of advanced workflows and techniques.

---

## Objectives

The repository was created to help developers, technical teams, and professionals:

- Better understand the practical use of Claude Code in real contexts;
- Improve their daily workflow;
- Explore techniques for automation, debugging, refactoring, and orchestration;
- Collect useful patterns, recurring cases, and replicable practices;
- Maintain an updated overview of features, operational possibilities, and new usage modes.

The approach is intentionally concrete: less abstract theory, more observable, testable, and reusable techniques.

---

## Course Structure

The material is organized into **14 modules**, covering the entire path of the **99 techniques**:

1. **Essential Commands**  
   Operational foundations, navigation, and context management.

2. **Productivity Techniques**  
   Methods to reduce friction, speed up work, and achieve more consistent results.

3. **Architecture Techniques**  
   Structural analysis, planning, and design support.

4. **Workflow Automation**  
   Hooks, pipelines, scripts, and recurring automations.

5. **Debug and Recovery**  
   Diagnosis strategies, problem reproduction, and recovery from incorrect implementations.

6. **Advanced Techniques**  
   Complex operational modes, parallelism, and high-intensity approaches.

7. **Deep Workflows**  
   Context management on long sessions, handoffs, worktrees, and multi-step processes.

8. **Hidden Power**  
   Less evident features, optimizations, and high-impact configurations.

9. **Community Techniques**  
   Patterns and practices emerged from real use.

10. **Advanced Architecture**  
    Extended refactoring, migrations, and structural reorganizations.

11. **Thinking Control**  
    Techniques to better guide analysis, reasoning, and task decomposition.

12. **CLI Mastery**  
    Advanced terminal use, scripting, and integration with external tools.

13. **System Secrets**  
    Limits, precautions, security, and less documented operational aspects.

14. **Last Level**  
    Advanced orchestration, agent coordination, and complex workflows.

## Project Philosophy

The idea behind this collection is simple: today it is not enough to know how to use a code generation or assistance tool, you need to know how to:

* Set up the work well;
* Structure the context;
* Guide the process;
* Verify results;
* Make the techniques that work repeatable.

For this reason, this repository is not intended to be just a showcase for the course, but a practical point of reference for those who want to deepen Claude Code with a more operational, technical, and progressively updatable approach.
